# Product Information Web Scraper 🛒

## Overview

This project is a Python-based web scraper that extracts product information from an e-commerce website and stores the collected data in a CSV file. The scraper gathers product names, prices, and ratings automatically.

## Features

* Extracts product names
* Extracts product prices
* Extracts product ratings
* Stores data in CSV format
* Automated data collection

## Technologies Used

* Python 3
* Requests
* BeautifulSoup4
* CSV Module

## How It Works

1. Sends a request to the target website.
2. Retrieves the HTML content.
3. Parses the webpage using BeautifulSoup.
4. Extracts product details such as name, price, and rating.
5. Saves the collected information into a CSV file.

## Skills Demonstrated

* Python Programming
* Web Scraping
* HTML Parsing
* Data Extraction
* File Handling
* CSV Data Storage

## Output

The program generates a CSV file containing product information in a structured format.

## Learning Outcome

This project helped me understand web scraping techniques, HTML structure, data extraction, and storing information in CSV files for further analysis.
